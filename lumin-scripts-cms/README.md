# Lumin Scripts CMS

Deno-based CMS data management system for **Strapi** and **Prismic** integrations.

---

## 🚀 Quick Start

### 1. Clone repository

```bash
git clone git@github.com:luminpdf/lumin-scripts-cms.git
cd lumin-scripts-cms
```

### 2. Install Deno

**macOS:**

```bash
brew install deno
```

**Windows (PowerShell):**

```powershell
irm https://deno.land/x/install/install.ps1 | iex
```

**Or with Chocolatey:**

```powershell
choco install deno
```

✅ Verify installation:

```bash
deno --version
```

---

### 3. Setup environment

Create a `.env` file in the project root:

```env
ENVIRONMENT=staging 
STRAPI_ENDPOINT=xxx
STRAPI_API_TOKEN=Bearer xxx
```
